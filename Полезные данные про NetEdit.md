# 🛣 Network Mode (F2) — Полное руководство

Network Mode — это режим редактирования дорожной сети в NetEdit. Здесь вы создаёте и изменяете дороги, перекрёстки, светофоры и всю инфраструктуру.

---

## 📁 Левая панель — иерархия сети

```
📂 Network
  ├── 📂 Edges (дороги)
  │    ├── edge_1
  │    ├── edge_2
  │    └── ...
  ├── 📂 Junctions (перекрёстки)
  │    ├── junction_1
  │    ├── junction_2
  │    └── ...
  ├── 📂 Traffic Lights (светофоры)
  │    ├── tls_1
  │    └── ...
  ├── 📂 Additional (доп. элементы)
  │    ├── busStops
  │    ├── detectors
  │    └── ...
  ├── 📂 Shapes (формы)
  │    └── полигоны, POI
  └── 📂 TAZs (транспортные зоны)
```

### Что можно делать в иерархии:
- **Клик** — выделить объект
- **Ctrl+Клик** — добавить к выделению
- **ПКМ** — контекстное меню объекта
- **Drag & Drop** — перемещать между папками (не для всех объектов)

---

## 🛠 Инструменты Network Mode (верхняя панель)

### 1. **Select** (S) — Выделение
Базовый инструмент для выбора объектов.

| Действие | Результат |
|----------|-----------|
| Клик по дороге | Выделить дорогу |
| Клик по перекрёстку | Выделить перекрёсток |
| Ctrl+Клик | Добавить к выделению |
| Рамка | Выделить всё в области |
| Shift+Рамка | Добавить область к выделению |

**Применение:** 
- Выделить несколько дорог для удаления
- Выделить область для массового изменения свойств

---

### 2. **Move** (M) — Перемещение
Перемещает узлы (перекрёстки) и геометрические точки.

| Действие | Результат |
|----------|-----------|
| Перетащить перекрёсток | Переместить со всеми дорогами |
| Перетащить гео-точку дороги | Изменить форму дороги |
| Ctrl+Перетащить | Ограничить движение по горизонтали/вертикали |

**Пример:** Исправить кривой перекрёсток или выровнять дорогу.

---

### 3. **Add Edge** (A) — Добавить дорогу
Создаёт новую дорогу между перекрёстками.

**Пошагово:**
1. Нажмите `A` (или иконку)
2. Кликните на начальный перекрёсток
3. Кликните на конечный перекрёсток
4. В инспекторе настройте дорогу:
   - `numLanes` — количество полос
   - `speed` — максимальная скорость (м/с)
   - `allow` — какие ТС могут ехать
   - `width` — ширина полосы (по умолчанию 3.2 м)

**Пример:** Добавить дублёр для разгрузки магистрали.

---

### 4. **Add Junction** (J) — Добавить перекрёсток
Создаёт новый узел (пересечение дорог).

**Пошагово:**
1. Нажмите `J`
2. Кликните в любом месте (или на существующую дорогу)
3. В инспекторе настройте:
   - `type` — `traffic_light` (светофор) / `priority` (главная дорога) / `unregulated` (нерегулируемый)
   - `radius` — радиус перекрёстка (для отрисовки)

**Особенность:** Если кликнуть на середину дороги, NetEdit автоматически разобьёт дорогу и создаст перекрёсток.

---

### 5. **Add Connection** (C) — Добавить соединение
Управляет тем, какие повороты разрешены на перекрёстке.

**Пошагово:**
1. Нажмите `C`
2. Кликните на исходную полосу (начала дороги)
3. Кликните на целевую полосу (конечной дороги)
4. Соединение создано!

**Важные параметры в инспекторе:**
| Параметр | Значение |
|----------|----------|
| `pass` | Разрешить движение прямо |
| `left` | Разрешить левый поворот |
| `right` | Разрешить правый поворот |
| `turn` | Разрешить разворот |
| `partially` | Частичное соединение (не для всех полос) |

**Пример:** Запретить левый поворот на загруженном перекрёстке.

---

### 6. **Add Traffic Light** (T) — Добавить светофор
Превращает обычный перекрёсток в регулируемый.

**Пошагово:**
1. Выделите перекрёсток
2. Нажмите `T` (или иконку светофора)
3. В инспекторе настройте фазы:
   - `programID` — имя программы (обычно "0")
   - `phases` — список фаз
     - `duration` — длительность фазы (с)
     - `state` — состояние полос (`G`=зелёный, `r`=красный, `y`=жёлтый)

**Пример фазы:**
```
state: GGGgrrrrrr
duration: 30
```
- Первые 4 полосы — зелёный
- Остальные — красный

---

### 7. **Add Crossing** (X) — Добавить пешеходный переход
Создаёт зебру через дорогу.

**Пошагово:**
1. Нажмите `X`
2. Кликните на дорогу (начало перехода)
3. Кликните на другую сторону дороги
4. В инспекторе настройте:
   - `width` — ширина перехода (м)
   - `priority` — приоритет для пешеходов

---

### 8. **Add Additional** (D) — Добавить доп. элементы
Самый разнообразный инструмент.

**Доступные типы:**

| Тип | Что делает | Где используется |
|-----|------------|------------------|
| **busStop** | Автобусная остановка | На краю дороги |
| **inductionLoop** | Индукционная петля (счётчик) | В асфальте |
| **laneAreaDetector** | Детектор зоны | Участок дороги |
| **rerouter** | Перенаправитель | На перекрёстке |
| **parkingArea** | Парковка | У дороги |
| **chargingStation** | Зарядка для EV | У дороги |

**Пример добавления busStop:**
1. Нажмите `D`
2. Выберите `busStop`
3. Кликните на дорогу
4. В инспекторе укажите:
   - `id`: `stop_1`
   - `length`: `15` (метров)
   - `lines`: `155,294` (какие маршруты)

## 🖱 Контекстное меню (ПКМ по полосе)

Вот что означают пункты, которые вы видели на скриншотах:

### Основные действия:

| Пункт | Что делает | Когда использовать |
|-------|------------|---------------------|
| **Center** | Центрирует вид | Быстро найти полосу |
| **Copy parent edge name to clipboard** | Копирует ID дороги | Для вставки в маршрут |
| **Copy name to clipboard** | Копирует ID полосы | Для детекторов |
| **Add Lane To Selected** | Добавляет полосу к выделению | Массовое редактирование |
| **Remove Edge From Selected** | Убирает дорогу из выделения | Очистка выделения |
| **Show Parameter** | Показывает параметры | Просмотр скрытых атрибутов |

### Операции с ребром (Edge operations):

| Пункт | Что делает | Пример |
|-------|------------|--------|
| **Split edge here** | Разрезает дорогу в точке | Создать пересечение |
| **Split edge in both directions** | Разрезает в обе стороны | Для двусторонней дороги |
| **Set geometry endpoint here** | Устанавливает конечную точку | Изменить форму дороги |
| **Select reachable** | Выделяет достижимые дороги | Проверить связность сети |
| **Reverse edge** | Меняет направление | Ошиблись при создании |
| **Add reverse direction** | Добавляет встречную полосу | Из односторонней сделать двустороннюю |
| **Straighten edge** | Выпрямляет дорогу | Убрать лишние изгибы |
| **Smooth edge** | Сглаживает форму | Красивая трассировка |

### Операции с полосой (Lane operations):

| Пункт | Что делает | Результат |
|-------|------------|-----------|
| **Duplicate lane** | Копирует полосу | Добавляет параллельную полосу |
| **Set custom lane shape** | Ручная форма | Для расширений/сужений |
| **Add restricted lane** | Добавляет ограниченную полосу | Выделенная полоса для автобусов |
| **Transform to restricted lane** | Превращает в ограниченную | Сделать полосу только для автобусов |

## 🔧 Параметры дороги (Edge) в инспекторе

При выделении дороги справа появляются эти параметры:

| Параметр | Тип | Что значит | Пример |
|----------|-----|------------|--------|

| **numLanes** | число | Количество полос | `2` |

| **width** | число | Ширина полосы (м) | `3.2` |

## 📝 Практический пример: Создание автобусной полосы

1. **Нажмите `F2`** (Network Mode)
2. **Найдите дорогу**, где нужна автобусная полоса
3. **Выделите дорогу** (клик)
4. В инспекторе **увеличьте numLanes** на 1 (например, с 2 до 3)
5. **Выделите новую полосу** (обычно крайняя правая)
6. **ПКМ → Transform to restricted lane**
7. В диалоге выберите `bus` и `taxi`
8. Нажмите **OK**

**Готово!** Теперь по новой полосе могут ехать только автобусы и такси.

## 💡 Советы для Network Mode

1. **Всегда сохраняйте** (`Ctrl+S`) перед большими изменениями
2. **Используйте `Ctrl+Z`** для отмены ошибок
3. **Проверяйте связность** — убедитесь, что все дороги соединены
4. **Для больших сетей** используйте `Select reachable`, чтобы найти изолированные участки
5. **Светофоры сложны** — начните с `type="priority"`, потом добавляйте светофоры

📊 Demand Mode (F3) — Полное руководство

Demand Mode — это режим создания транспортного спроса в NetEdit. Здесь вы создаёте автомобили, автобусы, маршруты, потоки и пешеходов.

text

📂 Demand
  ├── 📂 Vehicle Types (типы ТС)
  │    ├── DEFAULT_VEHTYPE
  │    ├── pt_bus
  │    └── ...
  ├── 📂 Routes (маршруты)
  │    ├── route_155
  │    ├── route_294
  │    └── ...
  ├── 📂 Vehicles (отдельные ТС)
  │    ├── veh_1

       └── ...

Что можно делать:

    Клик — выделить объект

    Двойной клик — отцентрировать вид на объекте

    ПКМ — редактировать/удалить

    Drag & Drop — скопировать тип ТС

🛠 Инструменты Demand Mode
1. Add Vehicle Type (F8) — Тип транспортного средства

Создаёт класс ТС с общими параметрами.
Параметры vType:
Параметр	Значение	Пример
id	Уникальное имя	city_bus
vClass	Класс ТС	bus / passenger / truck
length	Длина (м)	12.0 (автобус), 4.5 (легковой)
width	Ширина (м)	2.5

accel	Ускорение (м/с²)	1.2
decel	Замедление (м/с²)	4.5
maxSpeed	Макс. скорость (м/с)	20.0 (72 км/ч)
sigma	Случайность (0-1)	0.5 (чем выше, тем хаотичнее)
tau	Время реакции (с)	1.0
color	Цвет (RGB)	0,255,0 (зелёный)
emissionClass	Класс выхлопа	HBEFA3/PC_G_EU4

yaml

yaml

2. Add Route (F7) — Маршрут

Задаёт последовательность дорог, по которым будет ехать ТС.
Пошаговое создание маршрута:

    Нажмите F7

    Кликайте на дороги в нужном порядке

    Нажмите "Finish route creation" (внизу)

Альтернативный способ:

    Нажмите F7

    В инспекторе вручную введите edges через пробел

    Нажмите Enter

Параметры маршрута:
Параметр	Что значит	Пример
id	Уникальное имя	route_155
edges	Последовательность рёбер	edge1 edge2 edge3
color	Цвет отображения	255,154,210
Пример из вашего файла:
xml

<route id="pt_bus_155:0" edges="295298101#0 295298101#1 131700456#0 23265254#0 23265254#1 23265254#2 23265254#3 23265254#5 23265254#6 430045512#0"/>

3. Add Vehicle (F5) — Одиночное ТС

Создаёт одно транспортное средство для точного тестирования.
Параметры vehicle:
Параметр	Что значит	Пример
id	Уникальное имя	test_car_1
type	Тип ТС	passenger_car
route	Маршрут	route_155
depart	Время появления (с)	0

🔄 Связь между режимами Network и Demand
Network Mode	Demand Mode
Создаёт дороги (edges)	Создаёт маршруты из дорог
Создаёт перекрёстки	Определяет начальную/конечную точки
Определяет скорость на дороге	Тип ТС должен соответствовать разрешённым на дороге
Создаёт автобусные остановки	Остановки используются в маршрутах

Важно: Если в Network Mode дорога запрещена для автобусов (disallow="bus"), то автобус не сможет по ней ехать!
💡 Советы для Demand Mode

    Сначала создайте vType — потом используйте его везде

    Используйте flow вместо vehicle — для больших симуляций

    Проверьте совместимость — тип ТС должен быть разрешён на всех дорогах маршрута

    Останавливайте автобусы — обязательно добавляйте <stop> для реализма

    Тестируйте с одним vehicle — прежде чем создавать flow

# 📝 Пошаговые инструкции для NetEdit

## 🛣 1. Создание дороги (Edge)

### Способ 1: Между существующими перекрёстками

| Шаг | Действие | Результат |
|-----|----------|-----------|
| 1 | Нажмите `F2` (Network Mode) | Переход в режим сети |
| 2 | Нажмите `A` (Add Edge) | Активация инструмента |
| 3 | Кликните на **начальный перекрёсток** | Появится линия |
| 4 | Кликните на **конечный перекрёсток** | Дорога создана |
| 5 | В инспекторе (справа) настройте параметры | - |

### Параметры в инспекторе:

```yaml
numLanes: 2          # количество полос
speed: 13.89         # макс. скорость (м/с) = 50 км/ч
allow: passenger bus # кто может ехать
width: 3.2           # ширина полосы
```

### Способ 2: Создание с новыми перекрёстками

| Шаг | Действие |
|-----|----------|
| 1 | Нажмите `J` (Add Junction) |
| 2 | Кликните в любом месте — создан первый перекрёсток |
| 3 | Снова нажмите `J` |
| 4 | Кликните в другом месте — создан второй перекрёсток |
| 5 | Нажмите `A` и соедините их |

## 🔴 2. Создание перекрёстка (Junction)

### Новый перекрёсток:

| Шаг | Действие |
|-----|----------|
| 1 | Нажмите `F2` (Network Mode) |
| 2 | Нажмите `J` (Add Junction) |
| 3 | Кликните в нужном месте |
| 4 | В инспекторе настройте тип |

### Типы перекрёстков в инспекторе:

| Тип | Что значит | Когда использовать |
|-----|------------|---------------------|

| `traffic_light` | Со светофором | Загруженные перекрёстки |
| `unregulated` | Без приоритетов | Тихие улицы |
| `dead_end` | Тупик | Конец дороги |

### Как соединить дороги с перекрёстком:

| Шаг | Действие |
|-----|----------|
| 1 | Уже есть перекрёсток |

| 4 | Кликните на перекрёсток |
| 5 | NetEdit автоматически подключит |

| Шаг | Действие |
|-----|----------|
| 1 | Нажмите `F2` (Network Mode) |
| 2 | Нажмите `D` (Add Additional) |
| 3 | В выпадающем списке выберите **busStop** |
| 4 | Кликните на дорогу в нужном месте |
| 5 | В инспекторе настройте: |

```yaml
id: my_bus_stop
length: 15.0           # длина платформы (метров)
lines: 155 294         # какие маршруты останавливаются
friendlyPos: true      # автоматическая корректировка позиции
```

### Шаг 3: Создать маршрут с остановкой

| Шаг | Действие |
|-----|----------|
| 1 | Нажмите `F3` (Demand Mode) |
| 2 | Нажмите `F7` (Add Route) |
| 3 | Кликайте на дороги (включая ту, где остановка) |
| 4 | Нажмите **"Finish route creation"** |
| 5 | В инспекторе добавьте stop: |

```yaml
id: bus_route
edges: edge1 edge2 edge3
# Добавить stop можно в инспекторе через кнопку "+"
```

### Или вручную в XML после создания:

```xml
<route id="bus_route" edges="edge1 edge2 edge3">
    <stop busStop="my_bus_stop" duration="20.00" until="100.00"/>
</route>
```

### Шаг 4: Создать автобусный flow

| Шаг | Действие |
|-----|----------|
| 1 | `F3` → `F6` |
| 2 | Заполните: |

```yaml
id: bus_155
type: pt_bus
route: bus_route
begin: 0.00
end: 3600.00
period: 300.00        # каждые 5 минут
```

### Результат в XML:

```xml
<vType id="pt_bus" vClass="bus" length="12.0" width="2.5" accel="1.2" decel="2.5" maxSpeed="16.67" color="0,0,255"/>
<route id="bus_route" edges="edge1 edge2 edge3">
    <stop busStop="my_bus_stop" duration="20.00" until="100.00"/>
</route>
<flow id="bus_155" type="pt_bus" route="bus_route" begin="0" end="3600" period="300"/>
```

## 📋 Блок-схема последовательности действий

```
1. Network Mode (F2)
   ├── Создать перекрёстки (J)
   └── Создать дороги (A)

2. Demand Mode (F3)
   ├── Создать типы ТС (F8)
   ├── Создать маршруты (F7)
   │   └── Добавить остановки (в инспекторе)
   ├── Создать одиночные ТС (F5) — для теста
   └── Создать потоки (F6) — для симуляции
```

## ⚡ Быстрая шпаргалка

| Действие | Режим | Клавиша |
|----------|-------|---------|
| Создать перекрёсток | Network | `J` |
| Создать дорогу | Network | `A` |
| Создать тип ТС | Demand | `F8` |
| Создать маршрут | Demand | `F7` |
| Создать автомобиль | Demand | `F5` |
| Создать поток | Demand | `F6` |
| Создать остановку | Network | `D` → `busStop` |
| Сохранить всё | Любой | `Ctrl+S` |

---

Этих шагов достаточно для создания полной симуляции с дорогами, перекрёстками, автомобилями и автобусами!
